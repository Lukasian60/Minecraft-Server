Um anschließend den Server beitreten zu können, startest du das Spiel, gehst auf "Multiplayer", dann "add Server" und kopierst die Serveradresse (im Watsapp Kanal) in die Box wo drüber "Server adress" steht. Danach bennst du den Server "Bee_SMP", drückst auf "Done" und dann hast du Zugriff auf den Server!

Optional: Du kannst durch den Server (Auf den Server drauf -> esc -> Serverlinks) das Resourcepack für der Server runterladen.
