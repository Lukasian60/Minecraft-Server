Hir findet man Tutorials um mitspielen zu können.

Serverinformation: Minecraftversion 1.26.2 Fabric

-Modrinth download-

Um Anschließend den Server beitreten zu können Startest du das Spiel gehst auf "Multiplayer" dann "add Server" Kopierst die Serveradresse (im Watsapp Kanal) in die Box wo drüber "Server adress" steht. Danach bennst du den Server "Bee_SMP" drückst auf "Done" und dann hast du Zugriff auf den Server !

Optional: Du kannst durch den Server (Auf den Server drauf -> esc -> Serverlinks) das Resourcepack für der Server runterlanden
